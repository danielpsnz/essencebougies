# Essence Bougies e-commerce platform
E-commerce platform for my personal enterprise.

## Overview
This project is an ecommerce website where users can browse products, add them to a cart, and make purchases.

## Getting Started

### Prerequisites
- Node.js
- MongoDB or MySQL (depending on your choice)

### Installing

1. Clone the repo:
   ```bash
   git clone https://github.com/your-username/my-ecommerce-project.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the backend and frontend servers:
   ```bash
   npm start
   ```
